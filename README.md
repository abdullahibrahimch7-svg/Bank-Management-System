# Bank Management System

## Overview

This is a simple Bank Management System built in Python. The program allows users to perform basic banking operations through a menu-driven interface. This project was built as part of my Python learning journey to practice variables, conditions, user input, f-strings, and program logic.  

## Features

* Check account balance
* Deposit money
* Withdraw money
* Prevent withdrawal if balance is insufficient
* Exit the program safely

## Technologies Used

* Python
* If-Else Statements
* Loops
* Variables
* User Input

## How to Run

1. Download the Python file.
2. Open it in your Python IDE or terminal.
3. Run the program.
4. Choose an option from the menu.

## Example Menu

1. Balance
2. Deposit
3. Withdraw
4. Exit
